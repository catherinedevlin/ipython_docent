IPython Docent

An instructor's tool for use with ipython_doctester

  1. Instructor should set ipython_doctester.workshop_name and ipython_doctester.student_name in an IPython Notebook session, and execute each exercise once (success/failure is not important; this records each function name in the database in order).
  2. Results can be viewed at http://ipython_docent.appspot.com/workshop_name
  3. Each student should then set ipython_doctester.workshop_name to the same value, so that results will be visible in the same workshop_name "namespace". student_name should be unique within a workshop_name.
